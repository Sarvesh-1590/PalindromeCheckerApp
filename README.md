# Palindrome Checker App

A comprehensive Java application that demonstrates 13 different ways to validate palindromes, ranging from basic string manipulation to advanced design patterns and data structures.

## Project Structure

All use cases are located in the `App/src` directory as standalone Java applications:

1.  **UC1: Application Entry** - Welcome message and basic setup.
2.  **UC2: Hardcoded Validation** - Simple symmetric comparison.
3.  **UC3: Sentence Validation** - Handling spaces and case.
4.  **UC4: Character Array** - Using low-level arrays.
5.  **UC5: Stack-Based** - Using LIFO property of stacks.
6.  **UC6: Queue & Stack** - Comparing FIFO vs LIFO.
7.  **UC7: Deque-Optimized** - Bidirectional validation.
8.  **UC8: Linked List** - Using custom nodes and pointers.
9.  **UC9: Recursive Checker** - Logic through recursion.
10. **UC10: Normalized Check** - Regex-based preprocessing.
11. **UC11: OO Service** - Encapsulation using a Service class.
12. **UC12: Strategy Pattern** - Dynamic algorithm switching.
13. **UC13: Performance Comparison** - Benchmarking using `nanoTime()`.

## How to Run

Navigate to the project root and use the following commands:

### Compile all use cases:
```powershell
javac App/src/*.java
```

### Run a specific use case (e.g., UC13):
```powershell
java -cp App/src UseCase13PalindromeCheckerApp
```
